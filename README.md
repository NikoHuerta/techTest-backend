# RestServer Typescript Backend + MongoDB

Rebuild node modules ```npm install```\
Complete and rename file ```example.env``` to ```.env```


### Documentation API
Swagger Docs -> ```http://URL:PORT/api-docs```

### Develop
``` tsc --watch ```\
``` npm run dev ```

### Test
``` npm run test ```

### Build
``` npm run build ```

### Start Prod Build
``` npm start ```
